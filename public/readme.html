OddsJunction Project

Sports Betting Odds Tracker for Australian Sports

Overview

OddsJunction is a cloud-native application that automatically collects, processes, and displays betting odds for Australian sports including AFL, NRL, and Cricket. It uses AWS services for scalability, reliability, and automation — ensuring the latest odds are fetched every 6 minutes and made accessible via a responsive web interface.

Component Roles

  • VS Code: Local development environment.
  • GitHub: Code repository & version control.
  • CodePipeline: CI/CD for automatic deployments.
  • Elastic Beanstalk: Hosts app, runs cron jobs, serves frontend/API.
  • The Odds API: External API for live odds.
  • AWS DynamoDB: Stores odds history with TTL cleanup.
  • CloudFront: CDN for fast content delivery.
  • Route 53: DNS routing for the domain.
  • Certificate Manager: SSL/TLS management.
  • Users: Visitors using the OddsJunction site.
  • CloudWatch: Monitoring and logging.
